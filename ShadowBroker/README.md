# Shadow Broker

[Shadow Broker](https://github.com/BigBodyCobain/Shadowbroker) is an open-source intelligence (OSINT) platform that provides real-time global telemetry and tactical overlays.

## Deployment

1.  Copy `example.env` to `.env` (if provided or create one).
2.  Deploy using Docker Compose:
    ```bash
    docker compose up -d
    ```

## Features

- **Real-time Feeds:** Intelligence from 60+ live data sources.
- **Tactical Overlays:** Global flight coverage, ship tracking, and more.
- **AI Agent Integration:** Built-in AI support for tactical analysis.

For more information, visit the [GitHub repository](https://github.com/BigBodyCobain/Shadowbroker).
