# Floci

[Floci](https://floci.io) is a fast, free, and open-source AWS emulator. It allows you to develop and test your AWS applications locally without incurring any costs.

## Deployment

To deploy Floci, run:

```bash
docker compose up -d
```

## Features

- **Fast:** Designed for speed and low resource usage.
- **Free:** Completely free and open-source.
- **AWS Compatible:** Emulates various AWS services like S3, SQS, Lambda, and more.

For more information, visit the [official documentation](https://floci.io) or the [GitHub repository](https://github.com/floci-io/floci).
