# DocuSeal

[DocuSeal](https://www.docuseal.com/) is an open-source alternative to DocuSign that allows you to create, sign, and manage digital documents securely.

## Deployment

1.  Deploy using Docker Compose:
    ```bash
    docker compose up -d
    ```

## Features

- **Document Signing:** Easy and secure digital signatures.
- **Template Management:** Create and reuse document templates.
- **Self-Hosted:** Maintain full control over your documents and data.

For more information, visit the [official documentation](https://docs.docuseal.com/) or the [GitHub repository](https://github.com/docusealco/docuseal).
